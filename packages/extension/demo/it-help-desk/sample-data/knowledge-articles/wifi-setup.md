# Wi-Fi Setup by Office Location

## London Office — 10 Finsbury Square

### Network Names

| Network           | Purpose                                              | Password Required      |
| ----------------- | ---------------------------------------------------- | ---------------------- |
| **Contoso-Corp**  | Corporate devices (auto-connects on Contoso laptops) | No — certificate-based |
| **Contoso-Guest** | Visitors and personal devices                        | Yes                    |

### Contoso-Guest Password

The guest Wi-Fi password changes monthly. Current password is posted on the reception desk and in the London kitchen noticeboard. Ask reception if you cannot find it.

### Troubleshooting

- If Contoso-Corp does not auto-connect, forget the network and reconnect. Your device certificate should handle authentication.
- If prompted for credentials on Contoso-Corp, restart your laptop — the certificate may need to refresh.

## General Wi-Fi Tips

- Always prefer **Contoso-Corp** on company devices — it's faster and more secure
- Do not connect personal devices to Contoso-Corp (they will be rejected)
- If you need to connect a personal device for work purposes, use Contoso-Guest
- Video calls work best on wired ethernet connections in meeting rooms
