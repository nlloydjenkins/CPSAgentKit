# Software Installation and Requests

## Pre-Installed Software

All Contoso laptops come with the following software pre-installed:

| Software                                                | Purpose                | Auto-Updates |
| ------------------------------------------------------- | ---------------------- | ------------ |
| Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams) | Office productivity    | Yes          |
| GlobalProtect VPN                                       | Remote network access  | Yes          |
| Microsoft Edge                                          | Web browser            | Yes          |
| Microsoft Defender                                      | Antivirus and security | Yes          |
| Adobe Acrobat Reader                                    | PDF viewing            | Yes          |
| 7-Zip                                                   | File compression       | No           |

## Requesting Additional Software

If you need software that is not pre-installed:

1. Check the **Approved Software List** below
2. If the software is on the approved list, you can install it from the **Company Portal** app on your laptop
3. If the software is NOT on the approved list, submit a request through IT Support

### Approved Software (Self-Service via Company Portal)

These can be installed directly from the Company Portal app without IT approval:

- Google Chrome
- Mozilla Firefox
- Slack (for external client collaboration only)
- Zoom (for external meetings only — use Teams for internal)
- Visual Studio Code
- Notepad++
- FileZilla (SFTP client)
- PuTTY (SSH client)
- Snagit (screen capture)
- Microsoft Power BI Desktop

### How to Install from Company Portal

1. Open the **Company Portal** app (search for it in the Start menu)
2. Browse or search for the application
3. Click **Install**
4. Wait for the installation to complete (you may need to restart your laptop)

### Requesting Non-Approved Software

If the software you need is not on the approved list:

1. Contact IT Support through the IT Help Desk agent in Teams
2. Provide: software name, version, business justification, and your manager's name
3. IT will review the request:
   - **Standard requests** (developer tools, productivity apps): typically approved within 2 business days
   - **Licensed software** (design tools, specialist applications): may take 5-10 business days while procurement processes the licence

## Updating Software

- Microsoft 365 and Windows updates are managed automatically — do not skip or defer them
- If prompted to restart for updates, restart within 24 hours
- If an update causes issues, contact IT Support immediately

## Removing Software

Do not uninstall Microsoft 365, GlobalProtect, Microsoft Defender, or any other pre-installed security software. Doing so violates the Contoso Acceptable Use Policy.

To remove other software, use **Settings** → **Apps** → **Installed apps** → select the application → **Uninstall**.

## Contact

For software installation help, contact IT Support through the IT Help Desk agent in Teams.
