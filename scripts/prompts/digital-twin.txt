# Digital Twin (Charlie Nunn) — evaluation prompts.
# Lines starting with `#` are comments. Blank lines separate conversation
# groups: each group starts a fresh Direct Line conversation, so follow-up
# turns within a group share memory with the agent.
#
# Directives (recognised inline as a turn):
#   [CHOOSE FIRST]            — send the first option offered in the previous agent reply.
#   [CHOOSE LAST]             — send the last option.
#   [CHOOSE 2]                — send the Nth option (1-based).
#   [CHOOSE: keyword]         — send the option whose text contains "keyword" (case-insensitive).
#   [CHOOSE BEST]             — ask the judge model to pick the most sensible
#                               option in context of the originating question.
#                               Requires --judge.
#   [EXPECT: text]            — annotates the PREVIOUS turn with a tester
#                               expectation. Not sent to the bot; passed to
#                               the judge so it scores against the intent
#                               (e.g. "refuses all three in voice; doesn't
#                               pick one").
